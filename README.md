Reusable Apps
===
The repository contains some reusable apps, apis that can be plugged into a project with minor changes.
Follow the guide lines below to give it a try.

Run
===

```bash
git clone https://github.com/nabilmoiun/drf-reusables.git
cd drf-reusables
docker compose up --build -d
```

Hit http://localhost:8000/api/docs/ for the organised swagger documentation.
